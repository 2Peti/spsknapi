# SK
Od užívateľa si pýtajte čísla, kým používateľ nezadá 2 celé císla, porovnajte ich a vypíšte na obrazovku, ktoré z nich je väčšie. Použite:
- `try-catch`
- formátovanie
- ternary operator
# HU

- `try-catch` blokkot
