## Comments
`#` - Commented out code, the code does not run
```py
#print("Abc")

```
_CTRL+K+C_ comment whole row

_CTRL+K+U_ uncomment whole row

## Rename variable
Go over the variable with the cursour and press _F2_, it will rename all the occurences

## Save

_CTRL+S_ save

## Run code
_F5_ run with debug
